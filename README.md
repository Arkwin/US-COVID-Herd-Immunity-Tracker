## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Arkwin/US-COVID-Herd-Immunity-Tracker.git
   ```
2. cd into script directory
   ```sh
   cd US-COVID-Herd-Immunity-Tracker
   ```
3. Run the script
   ```sh
   ./tracker.sh
   ```
